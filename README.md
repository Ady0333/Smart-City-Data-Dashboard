# Smart-City-Data-Dashboard
A responsive, interactive dashboard that provides real-time weather conditions, air quality index (AQI), pollution levels, and health recommendations for any city worldwide. Built using HTML, CSS, JavaScript, and Chart.js, with live data from the OpenWeatherMap API. Features include city search, auto-refresh, dynamic charts, and a clean modern UI.
